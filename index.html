Uso recomendado

Buenas prácticas para mantener la base ordenada.

  • Registrar primero el ítem presupuestado.
  • Usar siempre el mismo nombre para cada proveedor o responsable.
  • Registrar avances y pagos como movimientos separados.
  • No sobrescribir el histórico: editar solo cuando hubo error de carga.
